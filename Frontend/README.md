# WebSecura React

A modern, React-based security scanning platform that helps users identify vulnerabilities and security issues in websites.

## 🌟 Features

- **Security Scanning** - Comprehensive website security analysis
- **User Authentication** - Secure login and registration system
- **Scan History** - Track and review all your security scans
- **Admin Dashboard** - Complete administrative control panel
- **Performance Analysis** - Website performance metrics (coming soon)
- **Responsive Design** - Works seamlessly on all devices
- **Real-time Results** - Instant scan results with detailed breakdowns

## 🚀 Quick Start

```bash
# Install dependencies
npm install

# Start development server
npm start

# Open browser to http://localhost:3000
```

## 📋 Prerequisites

- Node.js (v14+)
- npm or yarn
- Flask backend running at https://websecura.onrender.com
